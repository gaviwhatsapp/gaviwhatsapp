# Gavi WhatsApp API — Examples

Ready-to-run examples for the [Gavi WhatsApp API](https://www.gaviventures.com/docs/api).

## Examples

| Example | Description | Language |
|---------|-------------|----------|
| [send-message-ts](./send-message-ts) | Send a text message | TypeScript |
| [send-message-python](./send-message-python) | Send a text message | Python |
| [broadcast-campaign](./broadcast-campaign) | Send personalized broadcast to a CSV list | TypeScript |
| [webhook-receiver](./webhook-receiver) | Express server that receives incoming WhatsApp messages | TypeScript |

## Prerequisites

1. Sign up at [gaviventures.com](https://www.gaviventures.com) and connect your WhatsApp Business Account
2. Go to **Settings > API Keys** and create an API key
3. Set the key as an environment variable: `export GAVIWHATSAPP_API_KEY=gv_your_key`
