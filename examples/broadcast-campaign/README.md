# Broadcast Campaign

Send a personalized template message to a list of recipients from a CSV file.

## Setup

```bash
npm install
```

## Configure

1. Edit `recipients.csv` with your phone numbers and variables
2. Change the `template` name in `index.ts` to your approved template

## Run

```bash
export GAVIWHATSAPP_API_KEY=gv_your_key
npm start
```
